# Claude Code로 하루 만에 만든 대한민국 정부조직도 인터랙티브 시각화

영국에는 [machineryofgovernment.uk](https://machineryofgovernment.uk)라는 사이트가 있다. 정부 부처, 장관, 산하기관, 예산 정보를 인터랙티브하게 탐색할 수 있는 시각화 도구다. 한국에는 왜 이런 게 없을까? 그래서 만들었다.

**[korea-gov-org.pages.dev](https://korea-gov-org.pages.dev)**

![대한민국 정부조직도](og-image.png)

> 비공식 프로젝트입니다. 정확한 정보는 [정부24](https://www.gov.kr/portal/orgInfo)를 참고하세요.

---

## 왜 만들었나

한국 정부 조직은 꽤 복잡하다. 19개 부, 6개 처, 19개 청, 5개 위원회. 이재명 정부 출범 후 대규모 조직 개편이 있었고 — 기획재정부가 재정경제부와 기획예산처로 분리되고, 환경부가 기후에너지환경부로 확대되고, 검찰청이 공소청과 중수청으로 나뉘는 등 — 전체 그림을 한눈에 파악하기가 어렵다.

정부24에 조직도가 있긴 하지만, 정적인 표 형태라 관계를 직관적으로 이해하기 힘들다. 영국 사이트처럼 클릭하고 탐색할 수 있는 인터랙티브 버전이 있으면 좋겠다고 생각했다.

## 어떻게 만들었나

Claude Code와 대화하면서 하루 만에 v0부터 v14까지 14번의 이터레이션을 거쳤다.

> 업데이트: 2026-04-07 기준 프로젝트는 v23까지 확장되었고, 공식 현행 기준(19부 6처 18청 6위원회), 딥링크 안정화, 데이터 검증 스크립트가 반영되어 있다.

### 프로토타입 → 완성까지의 여정

처음에는 단순한 다크테마 카드 레이아웃으로 시작했다. 사용자 피드백(= 내 피드백)을 반복하면서:

1. **"조직도 같아 보여"** → D3.js 레이디얼 트리로 전환
2. **"기관장 정보가 없어"** → Wikipedia에서 40명+ 프로필 사진과 이력 수집
3. **"예산도 보고 싶어"** → 예산 모드 추가 (노드 크기 비례 + 순위)
4. **"모바일에서 안 보여"** → 반응형 + 바텀시트 + auto-fit 줌
5. **"마우스 올리면 관계도가 보였으면"** → 호버 시 경로 하이라이트 + 툴팁

이 과정에서 가장 흥미로웠던 건, 영국 사이트를 직접 분석할 수 없었다는 점이다. JS로 렌더링되는 사이트라 크롤링이 안 됐다. 결국 "이런 느낌"이라는 설명만으로 인터랙션을 구현해야 했고, 오히려 우리만의 방식으로 풀어낼 수 있었다.

### 기술 선택

- **D3.js v7**: 레이디얼 트리, 줌, 원형 패킹. 프레임워크 없이 단일 HTML.
- **KRDS 참고**: 한국 정부 디자인 시스템의 색상 토큰과 Pretendard 폰트를 차용했다.
- **Cloudflare Pages**: 정적 사이트라 무료 호스팅 + Web Analytics.

의도적으로 빌드 스텝을 없앴다. `index.html` 하나에 CSS, 데이터, JS가 모두 인라인으로 들어있다. 85KB짜리 단일 파일. 열면 바로 된다.

## 주요 기능

### 레이디얼 차트

대통령을 중심에 놓고 49개 기관이 방사형으로 퍼진다. 프로필 사진이 있는 노드는 원형으로 잘려서 표시된다.

마우스를 올리면 해당 기관에서 대통령까지의 경로가 하이라이트되고, 나머지는 흐려진다. 커서 옆에 뜨는 툴팁에는 계층 경로, 업무 설명, 예산, 기관장 정보가 표시된다.

### 예산 모드

"예산 규모" 토글을 켜면:
- **레이디얼**: 노드 크기가 예산에 비례한다. 보건복지부(110.4조)의 원이 압도적으로 크다.
- **조직도**: 예산 내림차순 정렬 + 상대 비율 바. 1위부터 순위 배지가 붙는다.

### 모바일

768px 이하에서는 완전히 다른 UX:
- 레이디얼 차트가 auto-fit 줌으로 화면에 맞게 축소
- 노드 탭 → 하단 바텀시트 (호버 대신)
- 스와이프 위로 → 거의 전체화면 확장
- 범례는 접기/펼치기 토글

## 데이터는 어디서

웹 검색 에이전트 3개를 병렬로 돌려서 수집했다:
1. 위키백과 — 장관 프로필 사진, 인사 이력
2. 정책브리핑/나무위키 — 청장/처장/위원장 인선
3. 각 부처 공식 사이트 — URL, 조직도, 업무

예산 데이터는 2026년 정부 예산안 기준이지만, 일부 추정치를 포함한다. 명칭변경된 부처(산업통상부, 기후에너지환경부, 성평등가족부 등)의 새 도메인도 에이전트가 찾아냈다.

## 비공식이라는 점

이건 중요해서 여러 번 강조한다. 이 사이트는:

- 대한민국 정부가 운영하는 **공식 사이트가 아니다**
- 기관 구조, 인사, 예산이 **실제와 다를 수 있다**
- **개인이 공개 정보를 기반으로 제작**했다

탑바에 빨간색 "비공식" 배지를 상시 표시하고, OG 이미지에도 "UNOFFICIAL" 태그를 넣었다. 출처 모달에는 정부24 공식 링크를 안내한다.
